## folder structure
Here's the folder structure for our django project:

```
Stockmng/     # Root directory.
|- principal/        # Folder used to load the main app.
|- statis/           # Folder containing static files.
|- stockmng/         # Main app for the project.
|- template/         #Folder containing the html templates for all the apges.
|- data.py           # package for loading and verifying data.
|- db.sqlite3        # Sqlite database for the website.
|- manage.py         # main file to run, manage the apps.
|- stock_package.py  # package containning function for stock management.
|- test_stock_package.py  # tests for the stock package functions.
```
